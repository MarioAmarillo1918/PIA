Scala

Lenguaje híbrido: funcional + orientado a objetos.

¿Qué es Scala?

Scala combina programación funcional con orientación a objetos y corre sobre la JVM. Es ampliamente usado en sistemas distribuidos como Apache Spark.

Características principales

  • Corre en la JVM
  • Combina OOP y funcional
  • Tipado estático
  • Soporte nativo para concurrencia

Ejemplo en Scala


val numeros = List(1,2,3).map(_ * 2)
println(numeros)
